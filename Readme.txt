# Projet Cryptographie

Ce projet est une implémentation de chiffrement et déchiffrement basée sur le système de chiffrement CSS (Content Scramble System).

## Fonctionnalités

1. Implémentation de deux registres à décalage à rétroaction linéaire (LFSR) de tailles 17 et 25.
2. Tests pour vérifier que les LFSR génèrent tous les états possibles.
3. Implémentation du système de chiffrement CSS.
4. Fonctions pour chiffrer et déchiffrer un message.

## Utilisation

Exécutez le fichier `Projet_Cryptographie.py` et choisissez une option :

1. Test LFSR 17 : Cette option teste le LFSR de taille 17 pour vérifier qu'il génère tous les états possibles.
2. Test LFSR 25 : Cette option teste le LFSR de taille 25 pour vérifier qu'il génère tous les états possibles.
3. Chiffrer un message : Cette option chiffre un message en utilisant le système de chiffrement CSS.
4. Déchiffrer un message : Cette option déchiffre un message en utilisant le système de chiffrement CSS.

## Exigences

Ce projet est écrit en Python et ne nécessite aucune dépendance externe.
